Glossary

tenant, CBV, ViewSet, atomic transaction, idempotency.

tenant: one company data boundary.
CBV: class-based view for web HTML.
ViewSet: DRF class for API actions.
atomic transaction: all-or-nothing DB change set.
idempotency: repeated call does not duplicate effect.
